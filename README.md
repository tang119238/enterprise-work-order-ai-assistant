# Enterprise Work Order AI Assistant

面向企业工单场景的可审计 RAG + Agent 作品集项目。默认离线运行，不需要大模型密钥。

> 本仓库仅使用合成制度和虚构工单，与任何真实企业、客户或生产系统无关。

## MVP 目标

- Spring Boot 提供工单详情、条件分页和返工链路三个只读接口。
- FastAPI 与 LangGraph 编排制度检索和工单工具调用。
- BM25 返回可定位的制度引用，不允许模型自行编造引用。
- 统一模型网关支持离线模式及多个国内大模型平台。
- Docker Compose 一条命令启动全部服务。

完整启动方式、接口示例、评测结果和模型配置将在 MVP 实现完成后补充。
